Java Student Management System

A Java Swing-based academic management system for student result calculation, eligibility checking, recovery plan generation, user management, and report handling. This page provides a web-friendly demo so recruiters can quickly understand the core project logic.

Project Features

  • Student information management
  • Course result calculation
  • Pass and failed status checking
  • Academic eligibility checking
  • Recovery plan generation
  • User login and management
  • CSV-based data reading and writing

How to Run the Java App

Clone this repository and open it with NetBeans.

Main class:

apu.group.java.by.kevin.Main

Required CSV files should be placed inside the data/ folder in the project root.

Note: The Web Demo is a browser-based visualization of the Java project's core calculation and recovery-plan logic. The original application is a Java Swing desktop system.
